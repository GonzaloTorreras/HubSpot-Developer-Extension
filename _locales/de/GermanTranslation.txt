Eine Chrome / Chromium-Erweiterung für HubSpot-Entwickler, die hilfreiche Verknüpfungen zu häufig verwendeten HubSpot-Abfrageparametern, Ressourcen und Tools enthält, um HubSpot Development einfacher und angenehmer zu gestalten.

* Schaltflächen zum Aktivieren von häufig verwendeten HubSpot-Abfrageparametern wie hsDebug, AMP, footer jQuery und einem Cache-Buster
* Google Page Speed ​​Insights Tester - großartig, um zu sehen, ob das Verschieben von jQuery in den footer eine signifikante Auswirkung hat
* Schnellzugriff auf die am häufigsten verwendeten Bereiche der HubSpot Dev Docs
* Optionales Dark Theme für den Design Manager
* Optionales Entwicklermenü integriert in HubSpot Nav
* Mac-Benutzer freuen sich! Wir haben das dunkle Cursor auf dunklen Code-Editoren Problem gelöst!
* Weitere Funktionen kommen!

Wenn Sie einen Fehler finden oder eine Funktion anfordern möchten, melden Sie es bitte in unseren GitHub-Issues. Wenn Sie an Beta-Tests neuer Versionen interessiert sind, können Sie sich auch unsere GitHub Seiten ansehen.
