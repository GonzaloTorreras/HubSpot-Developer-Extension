Une extension Chrome/Chromium pour HubSpot Développeurs qui ajoute des raccourcis utiles pour les paramètres de requête HubSpot couramment utilisés, des ressources et outils pour rendre le développement de HubSpot plus facile et agréable.

*Des boutons pour activer les paramètres de requête HubSpot couramment utilisés comme par exemple hsDebug, AMP, footer jQuery, ou le cache buster
*Le Testeur de Vitesse de Page Google - agréable de remarquer si bouger jquery dans le footer a un impact important
*Des raccourcis d'accès rapide aux zones les plus utilisées des Dev Docs HubSpot
*Un thème sombre optionnel pour le Design Manager
*Un menu developpeur optinnel intégré dans HubSpot Nav
*Les utilisateurs Mac se réjouissent ! Nous avons résolu le problème du curseur noir dans l'éditeur de code sombre
*Encore plus à venir !

Si vous trouvez un bug ou voulez reporter un problème, signalez-le dans nos Problèmes ("Issues") GitHub. Regardez notre GitHub également si vous êtes intéressés pour beta-tester des nouvelles versions.
